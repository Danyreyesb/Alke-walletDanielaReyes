# Alke-walletDanielaReyes
Alke wallet es una aplicacion de  Billetera Digital, permite al usuario realizar depositos, enviar dinero y revisar los movimientos realizados.




alke-wallet/
│
├── html/
│   ├── index.html

│   ├── sendmoney.html      
│   └── transactions.html  
│
├── css/
│   └── styles.css          
│
├── js/
│   └── main.js            
│
└── README.md




Funcionalidades principales

✔ Navegación entre páginas HTML

✔ Saldo inicial automático

✔ Depósitos que aumentan el saldo

✔ Envíos que descuentan el saldo

✔ Validación de saldo insuficiente

✔ Registro de movimientos


 Almacenamiento de datos

La aplicación utiliza localStorage para guardar:

✔ Saldo del usuario

✔ Historial de movimientos





Cómo ejecutar el proyecto

✔ Descargar o clonar el repositorio:

✔ https://github.com/Danyreyesb/Alke-walletDanielaReyes.git

✔ Abrir la carpeta del proyecto

✔ Ingresar a la carpeta html

✔ Abrir index.html en el navegador

✔ No requiere servidor ni instalación adicional.
Los datos se conservan incluso al cerrar el navegador.
